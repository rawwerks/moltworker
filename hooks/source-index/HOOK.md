# Source Index Hook

Injects the dirpack source code index into every agent run as a bootstrap file.

## What It Does

- Reads `DIRPACK.md` from the skills or workspace directory
- Adds it to the bootstrap files list so the agent knows its own source code
- Auto-generated by a GitHub Action running `dirpack pack` on every push to main

## Enable

```bash
openclaw hooks enable source-index
```
