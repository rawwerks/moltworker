# Source Index

This skill provides the moltworker source code index to the agent at bootstrap.
The index is auto-generated by a GitHub Action running `dirpack pack -t 5000` on every push to main.

The DIRPACK.md file at the repo root contains a compressed, pipe-delimited map of the entire
moltworker codebase: directory structure, key files, types, interfaces, and function signatures.

This gives the agent awareness of its own infrastructure source code.
